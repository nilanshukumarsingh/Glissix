NPM Package Usage

This demo shows how a developer imports the glissix npm library to drive realistic, tactile physical motion on a custom website instead of using static CSS transitions.

Select Physical Material

Target (Center) 0, 0
Card Position 0, 0
Velocity (px/frame) 0.00, 0.00
Active Material LEATHER 
// How you use it in your website:
import { Glissix } from 'glissix';

const motion = new Glissix(0, 0);
motion.useMaterial('LEATHER');

// Inside pointerup/drag end:
motion.applyImpulse(vx, vy);
motion.setTarget(0, 0);

// In your animation loop:
const { x, y } = motion.step();
card.style.transform = 
  `translate3d(${x}px, ${y}px, 0)`;
